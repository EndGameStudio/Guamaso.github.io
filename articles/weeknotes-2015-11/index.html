Six Months at Bitovi

I had my six month checkin this week. The realization that it’s been six months was weird. Time flies, man.

The past six months have been amazing. My desire to learn more and improve is constantly satisfied and challenged. I get to work with some bad ass people working on some bad ass projects. I’m pretty excited that it’s only been six months, that means I’m having fun.

CanJS Control

I’m trying out this pattern where, instead of adding more features to a component, I add the new feature as a can.Control that is designed to work specifically with the component. Lately, we’ve been adding feature after feature to some components and they’re getting harder and harder to maintain.

Even after six months, I’m still learning new things about our open source tools like CanJS. It feels like this is a good solution but I can’t be certain until after it gets used for a bit.

My other thought was to build additional functionality for a component within another discrete component. Then, if this functionality is needed, all one has to do is add the other component and target the original component. I’m not sure how the linking part would work so I decided to stick with the can.Control for now. We’ll see how it works out.

CanJS and Steal Part 2

After researching other CanJS projects, I figured out my problem with getting fixtures to work in my art project. Using the new release of CanJS, I also had to update the paths in the Steal config file. Changing the paths meant updating every file I had old paths in. Once this was done, everything worked GREAT.

Now, I’m having problems with my routes, unrelated to the above though. I’m still getting familiar with how to write solid routes so it’s a little hard to debug. Also, the build isn’t actually building (I’m using Steal to load dependancies in the browser), and I need to verify my routes are good. So the first step is to make sure everything builds nicely.

We’re heading to Austin this week for a Bitovi retreat (super excited!) so I’m going miss this weeks Code for Cary Meetup. I’m going to try and get a build working so we can have a prototype site running by the end of the week.
