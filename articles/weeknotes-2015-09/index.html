CanJS and StealJS

I got a chance to work on the Art Finder project for Code for Cary and I thought I would continue getting prototypes working.

For module loading, I am using StealJS. It’s the second time I’ve used it in a project with CanJS where I am using fixtures. I’m a little behind on the two tools as far as how they work with each other. The biggest problem is StealJS chokes on can.fixture.js because CanJS 2.1.4 doesn’t work with newest StealJS versions.

I’m really bummed out by this and surprised. The solution is to use the CanJS minor branch but that causes other errors in my code and fixtures still won’t load. This app won’t be live any time soon so there’s no real rush but it’s a little frustrating.

We’ll see how that goes.
