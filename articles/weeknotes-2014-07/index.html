Wakanda Training

At the beginning of the week, I attended a two day training at work on Wakanda. The two days reminded me of what they say about Calculus: it’s like 90% Algebra. Wakanda training (Wakanda being based on a JavaScript stack) was 90% JavaScript. It was a good refresh for myself and a very good one-oh-one-esque course for some of our “desktop” developers.

Sidebar: are they called “desktop” developers? I’ve heard them being referred to as “system language” developers. My point is, they are not web developers by trade. Writing code for the web can be very different. Throw in HTML markup and CSS styling and web applications get even tougher.

Anyway, the training was good. It was also good to hang out with the team for a while in RL.

Gulp and Grunt Meetup Talk Brief

I wrote up the brief, created my main bullet points and started working on the deck itself. I’m excited but I’m also very nervous. This is an interesting experiment - I don’t do public speaking. The Dev Journal’s goal was to improve how I learn. Having to publicize my thoughts forces me to think deeper about what I’m going to say. Doing a public speaking event, even something as humble as a local meetup, seems to be having the same effect but in multiples.

I submitted the brief and received a positive note. I’m waiting for the Meetup page to get updated before I put my nose to the grindstone on the presentation.

Fiddling with Firebase

I updated the code that feeds a random haiku on my home page. I’m trying to make it so I can link to specific haiku’s I write. Learning a lot about refactoring and how fun it is to shoot yourself in the foot. :<

Signed up for Digital Ocean

I have a web server that runs WordPress but I am not a fan of cPanel. It seems that I have come to prefer command line and direct access to the server. I heard about Digital Ocean from the Wakanda training and immediately purchased a droplet. For five dollars, I am able to purchase a virtual server with not-so-bad specs. Once the shared web server is up, I will bump up the specs in order to host WordPress and a couple of Ghost instances.

I’m writing up a list of what I did to get going with a droplet. I had about 2 days, cumulative, experience with running Linux through command line. There is a lot to write about from my experience. :)
