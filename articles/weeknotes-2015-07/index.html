CanJS scaffold

I spun up a project scaffold from scratch for the Code for Cary Art Finder Project. There isn’t really a good scaffolder for CanJS, unfortunately. There are a couple repos out there with Yeoman generators but they are a bit outdated in that they don’t work well with can.Component. That is to say, they work they just don’t generate blank components.

Either way, I thought it would be a good idea to put into practice what I have learned on CanJS project structure. I’m pretty happy with it and aim to post the project on Github soon.

I would like to actually keep this structure clean and make it into a repo that I can use later. I’ve never created a Yeoman generator but I hear they’re super easy. Might take a stab at that as well.
